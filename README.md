
# ADD+k Thesis Research Artifacts

This repository contains anonymized research artifacts for the thesis:

**Enhancing Architectural Decision Traceability in Attribute-Driven Design through Residuality Matrix Analysis**

Author: Aniia Kotomceva  
Institution: Innopolis University  
Year: 2026

## Purpose

The repository provides supplementary materials used in the empirical evaluation of the ADD+k method, an extension of Attribute-Driven Design that integrates residuality matrix analysis into the architectural design process.
